###################################
##Contents of smMIPs_panel Folder##
###################################

This folder contains everything needed for the smMIPs panel development.

1) input_files contain all files that were used to develop the panel coordinates

2) output_files contain all of the files generated from the codes that were used to develop the smMIPs capture panel

3) overlap_with_samples contains the script to elucidate the overlap of the smMIPs panel design with the available samples used in the analysis

4) scripts_for_probe_coordinates contains the python code used to pull informaiton from the CIViC interface as well as the R script used to create the coordinates for the smMIPs capture panel.
